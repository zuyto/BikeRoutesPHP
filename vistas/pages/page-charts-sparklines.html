jQuery Sparklines

A plugin for the jQuery javascript library to generate small sparkline charts directly in the browser.

Examples:
2,4,5,7,6,5,4,9,7,4,3,5,4,8,2,3,6,5,4,5,1,2,4,7,2,1,4,3,6,5,4,6
2,1,4,3,4,5,1,2,3,7,5,4,6
3,-2,4,-5,2,4,-3,-1
2,1,4,3,4,5,1,2,3,7,5,4,6
2,3,5
2,1,4
HTML Code: 
<div class="sparkline-line">2,4,5,7,6,5,4,9,7,4,3,5,4,8,2,3,6,5,4,5,1,2,4,7,2,1,4,3,6,5,4,6</div>
<div class="sparkline-bar">2,1,4,3,4,5,1,2,3,7,5,4,6</div>
<div class="sparkline-tristate">3,-2,4,-5,2,4,-3,-1</div>
<div class="sparkline-discrete">2,1,4,3,4,5,1,2,3,7,5,4,6</div>
<div class="sparkline-bullet">2,3,5</div>
<div class="sparkline-pie">2,1,4</div>
