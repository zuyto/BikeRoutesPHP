Clipboard.js

A modern approach to copy text to clipboard. No Flash. No dependencies. Just 3kb gzipped.

Basic Usage

First, you need to instantiate it by passing a DOM selector, HTML element, or list of HTML elements.

Here is a example of how we copy the examples codes on Bemat Admin.

Example: 
clipboard = new Clipboard('[data-copycode="true"]',{
	target: function(trigger) {
		return trigger.parentNode.parentNode.getElementsByTagName("pre")[0];
	}
});

clipboard.on('success', function(e) {
	$.toasts("add",{
		msg: "Copied!",
	});
	e.clearSelection();
});
