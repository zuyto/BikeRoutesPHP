i18next

i18next is a very popular internationalization framework for browser or any other javascript environment (eg. node.js).

Usage

Bemat Admin now supports translating your webapplication by using i18next javascript library.

You can activate the i18n support on bemat-config.js by setting the $.appMultiLanguage variable to true. This way Bemat will use a locale file placed on locales folder to translate your webapplication. Set the $.appDefaultLanguage to the name of the file that you want to use.

Translating Your WebApplication

In order to translate your webapplication, you need to add some data-i18n html attributes the html elements that you want to translate. Here is a example used in the Bemat sidebar.

<a href="pages/dashboard.html">
	<span class="menu-item-ico"><i class="material-icons">dashboard</i></span>
	<span class="menu-item-name" data-i18n="nav.dashboard">Dashboard</span>
</a>

Bemat will call automatically, after each new page load, the AppLanguage.doLocale() private function to apply the translation to the new active page.
