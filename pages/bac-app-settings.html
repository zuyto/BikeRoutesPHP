BEMAT App Settings

Config Bemat Admin to fit your application needs. You need html5 pretty links? No problem, Bemat support it!

App Settings
Note: These settings can be adjusted inside bemat-config.js file.
Name Type Default Description
$.debug boolean false Activates the debug console.
$.ajaxNavHtml5Mode boolean false Activates the html5Mode to the navigation. true will use pushstate instead of hash based navigation. With this option activated, you must edit your .htaccess file.
$.ajaxNavFadeDuration number 500 Sets the fade animation time of the page content.
$.ajaxNavMaterialTransition boolean false Activates the Material Design page transition. (Experimental)
$.appMultiLanguage boolean true Activates the multi language system.
$.appDefaultLanguage string en The code of the locale file used as default.
$.sidebarWidth number 300 The width of the sidebar. It must be the same width used in the CSS file to work properly.
$.sidebarMenuAutoCollapse boolean false Collapses the active open menu when you click on other menu item.
$.defaultHomePage boolean/string false Sets the default home page to load on app init.
$.idle boolean false Activates the idle mode timer.
$.idleTimeout number 3000 The timeout in miliseconds when the app will be on idle mode.
$.ckeditorSkin string lightwire Set the default CKEditor Skin.
$.prettifyTheme string github Set the default Google Prettify Highlight skin.
$.gMapsAPIKey string none Your Google Maps developer API key.
$.appBackgroundImage string binding_dark.png The page background image on Bxed Layout.
Bemat Admin API
Method/function Description
bematadmin.App.loadScript.load(); Use this function to load an external js file.
bematadmin.App.system.setUpComponents(); Use this function to reload the basic components.
bematadmin.App.navigate.go(); Use this function to go to an url using the Bemat navigation API
bematadmin.App.sidebar.toggle(); Use this function to show or hide the sidebar.
bematadmin.App.theme.darkHeaderFull(); Use this function to set to the dark style the full app header.
bematadmin.App.theme.darkHeaderBrand(); Use this function to set to the dark style just the header brand box.
bematadmin.App.theme.darkHeaderToolbar(); Use this function to set to the dark style just the header toolbar.
bematadmin.App.theme.darkSidebar(); Use this function to set to the dark style the sidebar.
bematadmin.App.theme.lightHeaderFull(); Use this function to set to the light style the full app header.
bematadmin.App.theme.lightHeaderBrand(); Use this function to set to the light style just the header brand box.
bematadmin.App.theme.lightHeaderToolbar(); Use this function to set to the light style just the header toolbar.
bematadmin.App.theme.lightSidebar(); Use this function to set to the light style the sidebar.
bematadmin.App.theme.sidebarCollapsed(); Use this function to collapse the sidebar.
bematadmin.App.theme.sidebarNormal(); Use this function to expand the sidebar.
Bemat Admin Events
Event Description
bemat-screen-resize This event is fired when the screen is resized.
idleOn This event is fired when the app goes into Idle Mode.
idleOff This event is fired when the app exits the Idle Mode.
scriptload This event is fired when the system scripts are fully loaded.
pageload This event is fired when the new page has been made visible to the user (will wait for CSS transitions to complete).
systemload This event is fired when the whole application is loaded and the splash screen has finished being hidden from the user.
