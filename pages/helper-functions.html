BEMAT Helper Functions

Bemat extends jQuery to make your development easier.

Helper Functions
Function Description
uniqId(); This function return a rand unique numbre id.
rand(min,max); This is a javascript clone of the rand php function.
typeOf(element) This function return the "type of" an element, variable, function or object.
$(element).toggleText(first text,second text); This function changes the text of an element on click event, from first text to second text and back.
$(element).exists(); This function return true if the element exists. Not chainable.
$(element).animateCss(animation name,callback); This function returns the callback when animate.css animation ends.
$(element).addAnimatedClass(class,callback); This function lets you add a class to an element and do the callback when the class transition ends.
$(element).removeAnimatedClass(class,callback); This function lets you remove a class of an element and do the callback when the class transition ends.
$(element).onCSSAnimationEnd(callback); This function listens until the css animation end, then do the callback.
$(element).onCSSTransitionEnd(callback); This function listens until the css transition end, then do the callback.
