BEMAT Helper Classes

Bemat helps you to build your own WebApp with these useful css classes.

Padding Helper CSS Classes
Class Description
.no-padding Removes all the padding of the element
.no-padding-horizontal Removes the left and right padding of the element
.no-padding-vertical Removes the top and bottom padding of the element
.no-padding-top Removes the top padding of the element
.no-padding-bottom Removes the bottom padding of the element
.no-padding-left Removes the left padding of the element
.no-padding-right Removes the right padding of the element

Padding Contextual Sizes

Class Description
.padding-xs Apply default small padding
.padding Apply default padding
.padding-lg Apply large padding

Padding Default Sizes

Class Description
.padding-{1-20} Apply default padding to all edges of the element. From 1 to 20 different sizes.
.padding-vertical-{1-20} Apply default padding top and bottom to the element. From 1 to 20 different sizes.
.padding-horizontal-{1-20} Apply default padding left and right to the element. From 1 to 20 different sizes.
.padding-top-{1-20} Apply default padding top to the element. From 1 to 20 different sizes.
.padding-bottom-{1-20} Apply default padding bottom to the element. From 1 to 20 different sizes.
.padding-left-{1-20} Apply default padding left to the element. From 1 to 20 different sizes.
.padding-right-{1-20} Apply default padding right to the element. From 1 to 20 different sizes.

Width, Height and Size Helper CSS Classes
Class Description
.auto-width Apply width:auto; style to the element.
.width-{1-20} Apply a default width to the element. From 1 to 20 different sizes.
.height-{1-20} Apply a default height to the element. From 1 to 20 different sizes.
.size-{1-20} Apply a default both width and height to the element. From 1 to 20 different sizes.
.full-size Apply 100% width and height to the element.
.max-width-100 Apply 100% max width to the element. If the element has a child img, it will inherit the style.
Flex Display Helper CSS Classes
Class Description
.flex-display Apply display:flex style to the element.
.flex-inline Apply display:inline-flex style to the element.
.flex Apply flex style to the element. With flex-grow and flex-shrink set to 1.
.flex-2 Apply flex style to the element. With flex-grow set to 2, and flex-shrink set to 1
.flex-100 Apply flex style to the element. With flex-grow and flex-shrink set to 0, and flex-basis set to 100%
.layout Apply display:flex style to the element.
.layout-column Apply flex-direction: column to the element.
.layout-row Apply flex-direction: row to the element.
.layout-align-space-between Apply justify-content: space-between to the element.
.layout-align-center Apply align-items: center to the element.
.layout-center-vertical Apply justify-content: center to the element.
Align Helper CSS Classes
Class Description
.vertical-align Centers the element on the Y axis, referent to its parent container.
.horizontal-align Centers the element on the X axis, referent to its parent container.
Opacity CSS Classes
Class Description
.opacity-0 Set opacity of the element to 0%.
.opacity-25 Set opacity of the element to 25%.
.opacity-50 Set opacity of the element to 50%.
.opacity-75 Set opacity of the element to 75%.
.opacity-100 Set opacity of the element to 100%.
Border CSS Classes
Class Description
.border-sm Add a border to the element.
.border-lg Add a border to the element.
.border-xl Add a border to the element.
.border-xxl Add a border to the element.
.border-white Add a white border to the element.
.border-gray Add a gray border to the element.
.border-transparent Add a transparent border to the element.
.hidden-border Add a transparent border to the element.
.border-dashed Add a dashed border to the element.
.bordered Add border to the element with hover state.
.border-circle Set a circle element
.border-rounded Add a rounded border to the element.
.no-radius Remove the radius or rounded border of the element.
Miscellaneous Helper CSS Classes
Class Description

.divider-vertical Add a vertical divider element
.text-overflow Element text can be clipped to fit it's width, displaying an ellipsis.
.no-scrollbar Element overflow hidden.
.no-scrollbar-x Element overflow-x hidden.
